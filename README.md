# EVEN-ODD

## ゲーム概要
1〜6 の番号に、それぞれ **0 または 1** がランダムに割り当てられます。  
プレイヤーは「偶数か奇数か」だけをヒントに、最終的に全ての番号の値を推理します。

## ルール
- 各ラウンドで **最大3つの番号** を選択できます。
- 選んだ番号に割り当てられた値 (0/1) を合計し、その合計が **偶数か奇数か** だけが表示されます。
- 選択できるのは **3つまで**。それ以上は選べません。
- この操作を **最大6ラウンド** 繰り返すことができます。
- 6ラウンド終了後、プレイヤーは 1〜6 の番号に割り当てられた値を **自分の推理で入力**します。
- 入力が正解かどうかが判定されます。

## 勝利条件
- 6つすべての番号について、割り当てられた 0/1 を正しく当てること。

## プレイの流れ
1. 番号を選択（最大3つ）
2. 「偶数チェック」で結果を確認
3. 自動的に次ラウンドへ進む
4. 5. 6ラウンド終了後、推理した答えを入力
6. 正解判定が表示される
